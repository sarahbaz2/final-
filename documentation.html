Visualization Documentation

General Overview

Netflix is one of the world’s leading streaming platforms, offering thousands of movies and TV shows across various genres and countries. Due to its popularity and global reach, Netflix has become a rich source of data for analysis and visualization.

In this project, I used the Netflix Titles Dataset, which includes metadata for every title on the platform, such as:

  • Type (Movie or TV Show)
  • Title
  • Director & Cast
  • Country of production
  • Release year
  • Rating
  • Listed_in (genres)
  • Description

While exploring the dataset, I became interested in understanding the distribution of Netflix content. This led to two main visualization ideas:

  • Visualization 1: Compare Movies vs TV Shows.
  • Visualization 2: Search and count titles by genre using the listed_in column.

These visualizations together help provide both a high-level overview of Netflix’s content and a deeper exploration into specific genres.

First Visualization: Movies vs TV Shows

Purpose

The purpose of the first visualization is to compare the total number of Movies and TV Shows available on Netflix using an interactive D3.js bar chart. The visualization answers the question: "Which type of content dominates Netflix?"

Dataset Column Used

This visualization uses only one column from the dataset:

  • type: indicates whether the title is a Movie or a TV Show.

Data Processing

After loading the CSV file, the data was processed by counting how many entries were Movies and how many were TV Shows. Then, percentages were calculated based on the total number of titles.

  • Count total Movies
  • Count total TV Shows
  • Compute total titles
  • Calculate percentage for each category

Visualization Design

The first visualization uses a bar chart created with an SVG canvas. It includes:

  • A band scale for the X-axis (Movie vs TV Show)
  • A linear scale for the Y-axis
  • Value labels above each bar
  • Dynamic title and subtitle
  • Netflix-inspired color palette (red and pink)

Interactivity

A toggle button allows users to switch between:

  • Count view: shows the total number of Movies and TV Shows.
  • Percentage view: shows the relative share each category contributes to the full dataset.

When switching modes, the bars, labels, and Y-axis values update smoothly.

Tooltip and Transitions

Hovering over a bar displays a tooltip showing:

  • Category (Movie or TV Show)
  • Count
  • Percentage

Smooth animations and transitions improve readability and user engagement.

Key Insights

Based on the dataset:

  • Movies make up the majority of Netflix content.
  • TV Shows represent a smaller portion.
  • Despite Netflix’s strong reputation for original shows, Movies still dominate numerically.

Second Visualization: Genre Popularity Over Time

Purpose

The purpose of this visualization is to explore how the popularity of specific genres has changed over the years on Netflix. This visualization allows the user to select any genre from the dataset and observe the number of titles within that genre released each year. It provides an interactive way to analyze long-term trends in Netflix content production.

How We Got the Idea

While examining the dataset, we noticed that each title includes a listed_in field, which contains one or more genres. This allowed us to analyze content categories across time. Because Netflix has continuously expanded its catalog, we wanted a visualization that shows how different genres grow or decline over time. This idea is useful for understanding audience trends, Netflix’s production strategies, and genre popularity shifts across years.

Steps We Followed to Build the Visualization

1. Extracting All Genres

We began by scanning the entire listed_in column to identify every unique genre. Many titles include multiple genres separated by commas, so we had to split each cell and collect all distinct values. The final result was a comprehensive list of all genres found in the dataset.

2. Creating an Interactive Genre Dropdown

After identifying all genres, we generated an interactive dropdown menu. This allows users to easily browse all available genres instead of manually typing them. The dropdown updates dynamically based on the dataset, ensuring it stays accurate.

3. Filtering the Dataset by Selected Genre

When the user selects a genre, the visualization filters all Netflix titles to find those that contain the chosen genre inside the listed_in field. This ensures that even titles with multiple genres are included correctly.

4. Grouping Titles by Release Year

For the filtered results, we counted how many titles were released each year. This produces a clear year-by-year timeline of the genre’s popularity. The dataset’s release_year field allowed us to create a consistent timeline.

5. Building the Line Chart

We used a D3.js line chart to represent how genre popularity changes over time. The X-axis represents years, while the Y-axis shows the number of titles released for the selected genre. A smooth animated transition makes the chart visually appealing and easy to interpret.

6. Adding Interactivity

The visualization updates automatically every time the user selects a different genre. The transition between states is animated, making it intuitive and engaging.

7. Interpretation of Results

This visualization allows users to explore questions such as:

  • Which genres grew the most in recent years?
  • How did Netflix expand into new content categories?
  • Are some genres consistently popular across time?

The result is a dynamic, insightful, and user-friendly visualization that complements the first visualization by adding a deeper temporal perspective.

Visualization 3: Top Countries by Releases

Design Process

The goal of this visualization is to explore which countries released the most content on Netflix in a given year. I started by sketching a horizontal bar chart layout on paper, considering how to handle multiple countries per title. Since many titles are listed in multiple countries, I decided to split the country column by commas and count each country separately. This allows the visualization to accurately reflect which countries are most active in releasing Netflix content.

Rationale of Design Choices

  • Chart Type: Horizontal Bar Chart – makes it easy to compare country counts side by side.
  • Interactive Dropdown: Users can select a release year. The chart updates dynamically to show the top countries for that year.
  • Color: A consistent fill color (#1f77b4) is used for all bars to keep focus on the data rather than the color encoding.
  • Sorting: The bars are sorted from highest to lowest count to immediately show the top countries.

How I Used the Visualization

By interacting with the dropdown to select different years, I was able to discover trends over time. For example, some countries consistently have high release counts, while others appear only in certain years. This helps identify which regions are most active in Netflix content production.
